STM32CubeWB Firmware Package V1.1.1 / 17-May-2019

Main Changes
  • Firmware Upgrade Services (FUS) V1.0.2:
    • Add support for STM32WB5xxE(512K) and STM32WB5xxC(256K) devices.
    • On STM32WB5xxC and STM32WB5xxE, it is mandatory to install FUS V1.0.2 before any other operation. Otherwise, the device might be locked in an unrecoverable state.
  • Wireless Coprocessor Binary:
    • BLE System: Fix stopMode2 race condition
    • BLE Security: Fix pairing issue with numeric comparison
    • Impact for stm32wb5x_BLE_Stack_fw.bin and stm32wb5x_BLE_HCILayer_fw.bin only.
  • STM32 WPAN:
    • Fix race condition in transport layer when an operating system is used.
    • Mesh Library V1.09.000
      • Light HSL Model implementation.
      • Sensor Server Model Example.
      • CID, PID Configuration.
      • Fix of the BD Address issue.
